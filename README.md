## ✨ Overview

**Infinity** is a premium, full-scale gym management application designed to bridge the gap between front-desk operations and executive oversight. Built with a stunning "liquid glass" and modern corporate aesthetic, it provides gym staff with instantaneous member lookup and check-in capabilities, while offering management deep, data-driven financial insights and automated report generation.

## 🚀 Key Features

- **Role-Based Access Control:** Distinct operating environments for Front Desk receptionists and Executive Managers.
- **Lightning-Fast Member Management:** Real-time member registration, subscription tracking, and instant check-ins via a unified global search mechanism.
- **Financial Architecture:** Centralized ledger for tracking incoming revenue (memberships, walk-ins, merchandise) and operational expenses.
- **Executive Dashboards:** Premium data visualization rendering real-time attendance trends, revenue distributions, and dynamic growth metrics.
- **Automated Reporting Engine:** Generate high-fidelity analytical reports directly into **PDF**, **Excel**, or dispatched instantly via formatted **HTML Emails**.
- **Mobile-First Infrastructure:** A flawlessly responsive UI featuring bottom app bars, off-canvas drawers, and unified quick-action dropdowns exclusively tailored for mobile usability.

## 🛠️ Technology Stack

Designed for maximum performance and fluid user experiences, Infinity leverages a modern, cutting-edge architecture:

### Frontend
* **React 18** + **Vite:** High-performance rendering and blistering fast hot-module replacement.
* **Tailwind CSS v4:** Utility-first styling powering the deep frosted glassmorphism, dynamic gradients, and precision responsiveness.
* **React Router v6:** Seamless single-page application routing and protected layout boundaries.
* **Lucide React:** Crisp, highly customizable geometric iconography.
* **Export Engines:** `jspdf` & `jspdf-autotable` (PDFs), `xlsx` (Excel Arrays), and `@emailjs/browser` (Networkless Email Transmission).

### Backend (BaaS)
* **Supabase:** Providing instant, real-time PostgreSQL database synchronization.
* **Supabase Auth:** Secure JWT-based authentication enforcing strict Row-Level Security (RLS) policies.
